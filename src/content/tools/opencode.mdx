OpenCode is an open source AI coding agent that helps developers write code directly from the terminal, IDE, or desktop application. It supports multiple AI providers like OpenAI, Claude, and Gemini, and includes built-in agents for development work and code analysis. The tool can execute commands, search files, modify code, and track file changes during sessions. With over 70,000 GitHub stars and 650,000 monthly users, OpenCode integrates with VS Code, Cursor, and other IDEs, offering features like session management, LSP support for code intelligence, and custom commands for predefined prompts.
