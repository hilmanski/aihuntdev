n8n is a workflow automation platform that allows developers to build AI-powered workflows and multi-agent systems. It combines visual drag-and-drop building with the ability to add custom code in Python or JavaScript, offering flexibility to create complex AI workflows with branching, loops, and human-in-the-loop controls. The platform integrates with over 500 apps and services, supports multiple LLMs, includes debugging tools with inline logs and data replay, and can be self-hosted or used in the cloud. It provides templates for common AI use cases and offers enterprise features like role-based access control and log streaming.
